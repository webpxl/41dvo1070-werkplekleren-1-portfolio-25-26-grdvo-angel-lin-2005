School Project

Scripting Project

Voor het vak Scripting kreeg ik de opdracht om een website te ontwikkelen voor een restaurantmenu. Mijn rol binnen dit project was het volledig zelf coderen van de website en het ontwerpen van een aantrekkelijke en overzichtelijke lay-out. Vervolgens presenteerde ik mijn code aan de leerkracht en deelde ik het project via GitHub, zodat alles transparant gevolgd en nagekeken kon worden. Ik heb de applicatie WebStorm gebruikt om deze website te maken met JavaScript als hoofddoel. Vervolgens heb ik ook HTML en CSS gebruikt om de website vorm te geven.

Hoofdpagina: De basis layout van het restaurantmenu met navigatie en menustructuur.

Verschil 1: Hier zijn interactieve buttons toegevoegd aan de menuitems, waardoor gebruikers items kunnen selecteren en toevoegen aan hun bestelling.

Verschil 2: Een volledig werkende winkelmand is geïmplementeerd, waar geselecteerde items worden verzameld met prijzen en totaalbedrag - dit toont de Javascript functionaliteit.

Eindresultaat

Het Eindresultaat: Dit is de volledige, werkende website van de scripting opdracht met alle functionaliteiten geïntegreerd. Een volledig functioneel restaurantmenu met interactieve elementen, winkelmandfunctionaliteit en een professionele gebruikersinterface - het bewijs van mijn Javascript, HTML en CSS vaardigheden!
